# Human UI Review

## Goal

Every interface must look handcrafted by an experienced product designer.

Never produce layouts that immediately look AI-generated.

Design with intention, hierarchy, rhythm, and consistency.

---

# Anti-AI Rules

Immediately reject:

• Huge rounded cards everywhere

• Random gradients

• Neon colors

• Oversized glowing shadows

• Glassmorphism abuse

• Giant floating icons

• Empty sections with no purpose

• Equal spacing on everything

• Too many animations

• Generic hero sections

• Fake statistics

• Stock illustrations

• Emoji icons

• Inconsistent border radius

• Random color palettes

• AI-generated marketing copy

---

# Layout Review

Check:

✓ Visual hierarchy

✓ Alignment

✓ Grid consistency

✓ Section rhythm

✓ White space

✓ Reading flow

✓ Content density

✓ Scroll experience

✓ Responsive behavior

Never leave large empty spaces.

Every section must have a purpose.

---

# Typography Review

Verify:

Heading hierarchy

Font consistency

Readable line-height

Letter spacing

Paragraph width

Maximum readability

Avoid:

Huge headings

Tiny body text

Too many font weights

Random capitalization

---

# Spacing Review

Check:

Section spacing

Component spacing

Card padding

Button spacing

Form spacing

Table spacing

Maintain an 8px spacing system.

# Color Review

Random accent colors

---

# Human Design Checklist

Every page must answer:

Does it feel intentional?

Does it feel premium?

Would this pass as work from a senior product designer?

Would this fit alongside:

- Linear

- Stripe

- Notion

- Attio

- Vercel

- Raycast

If not:

Redesign.

---

# Final Review

Score every page from 1–10:

Visual Design

UX

Accessibility

Responsiveness

Consistency

Performance

Professionalism

If any category scores below 9/10:

Provide detailed improvement tasks.

Never approve average designs.

Always push toward production-quality interfaces.
